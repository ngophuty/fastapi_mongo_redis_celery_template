## run command
```
    $ chmod +x ./scripts/*.sh
    $ ./scripts/initialize_project.sh
```
## install env
- Activate venv :
```
    $ source .venv/bin/activate
```
- install lib
```
    $ poetry install
```
- Run app
```
    $ python run_app.py
```
## run precommit
- cmd 
```
  $ pre-commit run --all-files
```
